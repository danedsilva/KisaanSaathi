# KisaanSaathi
Farmer friendly agriculture website

🌾 KisanSaathi — Farmer Welfare Portal
KisanSaathi is a multilingual Farmer Welfare Information Portal designed to help Indian farmers easily access government schemes, eligibility checks, documentation guides, and real-time updates in their own language.

This project simplifies access to central and Karnataka state agricultural schemes through an interactive, user-friendly web interface.

🚀 Features
📋 Scheme Explorer
Browse Central & Karnataka government schemes

View eligibility criteria

Document checklist

Direct application links

📡 Live PM-KISAN Guide (2026)
Includes:

Installment updates

Eligibility rules

Exclusion criteria

Required documents

Registration steps

✅ Eligibility Checker
Interactive question-based system that:

Identifies eligible schemes

Suggests benefits

Reduces confusion for farmers

🤖 AI Assistant
Chat-based assistant helps farmers:

Understand schemes

Navigate application steps

🌐 Multilingual Support

🧠 Problem Statement
Many farmers in India:

Don't know which schemes they qualify for

Don't understand application steps

Miss benefits due to lack of guidance

Face language barriers

KisanSaathi solves this by providing a single unified platform for scheme discovery and assistance.
